# sales-and-profit-analysis


# Description

This project focuses on building an interactive Power BI dashboard to analyze sales performance and profitability across multiple business dimensions.

# Key Features

1.KPI tracking for Total Sales and Total Profit

2.Monthly sales trend analysis to identify seasonality

3.Category and sub-category level performance insights

4.Customer-wise sales and profit contribution analysis

5.State-wise sales performance visualization

6.Interactive slicers for year and month filtering

# Tools & Technologies:

Power BI

DAX

Data Modeling

Data Visualization

# Business Value

The dashboard enables stakeholders to monitor performance, identify profit drivers, optimize product and customer strategies, and make data-driven business decisions.
